WS, WS/48, and offensive_win_shares, defensive_win_shares are very similar so we will only use one of them. Rate statistics are generally better than counting statsitics so we will use WS/48

Offensive_box_plus_minus does better than box_plus_minus and defensive_box_plus_minus so we will use offensive

We will use Win% instead total Team wins since there were a few seasons when not all teams played 82 games

pra_sum does better than PTS so we will use that
